# Experiment Details: 18CrNiMo7-6 Impact Tension

- Test type: impact tension
- Material: 18CrNiMo7-6
- Incident bar length: `1800 mm`
- Transmitted bar length: `1800 mm`
- Bar diameter: `20 mm`
- Specimen parallel-section length: `9.5 mm`
- Specimen diameter: `2.5 mm`
- Bar elastic modulus: `207 GPa`
- Bar density: `7800 kg/m^3`
- Incident-bar strain gauge distance to specimen end face: `839 mm`
- Transmitted-bar strain gauge distance to specimen end face: `877 mm`

## Software Input Guide

Applicable data files: `18crnimo7_6_steel/sample_01/raw_strain.xlsx` or `18crnimo7_6_steel/sample_02/raw_strain.xlsx`

1. Data import
- Click `Import CSV/XLSX` / `导入 CSV/XLSX`, then choose `raw_strain.xlsx`.
- `Time column` / `时间列`: select `time_s` or `Time/s`.
- `Incident bar column` / `入射杆列`: select the incident-bar strain column, such as `incident_strain/ε` or `incident_bar_signal/ε`.
- `Transmitted bar column` / `透射杆列`: select the transmitted-bar strain column, such as `transmitted_strain/ε` or `transmission_bar_signal/ε`.

2. Acquisition and preprocessing
- `Time unit` / `时间单位`: select `s`.
- `Strain unit` / `应变单位`: select `strain`, because this file already stores dimensionless strain.
- `Baseline correction` / `基线修正`: beginners can keep the default `start_mean`.
- `Filter method` / `滤波方式`: beginners can start with `none`; try `moving_average` only if the waveform is visibly noisy.

3. Bar parameters
- `Incident bar diameter mm` / `入射杆直径 mm`: `20`
- `Transmitted bar diameter mm` / `透射杆直径 mm`: `20`
- `Elastic modulus GPa` / `弹性模量 GPa`: `207`
- `Density kg/m3` / `密度 kg/m3`: `7800`
- `Incident gauge distance mm` / `入射应变片距端 mm`: `839`
- `Transmitted gauge distance mm` / `透射应变片距端 mm`: `877`

4. Specimen and sign convention
- `Section shape` / `截面形状`: select `cylinder`.
- `Specimen diameter mm` / `试样直径 mm`: `2.5`
- `Initial height/gauge length mm` / `初始高度/标距 mm`: `9.5`
- `Experiment type` / `实验类型`: select `tension`.
- `Sign convention` / `符号约定`: select `tension_positive`.

5. Processing and export
- Keep `Enable automatic fine alignment` / `启用自动微调对齐` checked.
- Do not check `Enable standard SHPB wave separation` / `启用标准SHPB波形分离`; this dataset uses the regular impact-tension three-wave/two-wave workflow.
- Click `Auto process` / `自动处理`.
- Review `Workflow status and result summary` / `流程状态与结果摘要`, `Three-wave alignment` / `三波对齐`, and `Results` / `计算结果`.
- If the results are reasonable, click `Export Excel` / `导出 Excel` or `Generate report` / `生成报告`.
