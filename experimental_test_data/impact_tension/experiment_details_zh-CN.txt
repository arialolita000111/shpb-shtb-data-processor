# 实验详情：18CrNiMo7-6 冲击拉伸

- 试验类型：冲击拉伸
- 材料：18CrNiMo7-6
- 入射杆长度：`1800 mm`
- 透射杆长度：`1800 mm`
- 杆件直径：`20 mm`
- 试样平行段长度：`9.5 mm`
- 试样直径：`2.5 mm`
- 杆件弹性模量：`207 GPa`
- 杆件密度：`7800 kg/m^3`
- 入射杆应变片至试样端面的距离：`839 mm`
- 透射杆应变片至试样端面的距离：`877 mm`

## 软件页面填写指南

适用数据文件：`18crnimo7_6_steel/sample_01/raw_strain.xlsx` 或 `18crnimo7_6_steel/sample_02/raw_strain.xlsx`

1. 数据导入
- 点击 `Import CSV/XLSX` / `导入 CSV/XLSX`，选择 `raw_strain.xlsx`。
- `Time column` / `时间列`：选择 `time_s` 或 `Time/s`。
- `Incident bar column` / `入射杆列`：选择入射杆应变列，例如 `incident_strain/ε` 或 `incident_bar_signal/ε`。
- `Transmitted bar column` / `透射杆列`：选择透射杆应变列，例如 `transmitted_strain/ε` 或 `transmission_bar_signal/ε`。

2. 采集与预处理
- `Time unit` / `时间单位`：选择 `s`。
- `Strain unit` / `应变单位`：选择 `strain`，因为该文件已是无量纲应变数据。
- `Baseline correction` / `基线修正`：新手建议先使用默认 `start_mean`。
- `Filter method` / `滤波方式`：新手建议先选 `none`；如果波形噪声明显，再尝试 `moving_average`。

3. 杆件参数
- `Incident bar diameter mm` / `入射杆直径 mm`：`20`
- `Transmitted bar diameter mm` / `透射杆直径 mm`：`20`
- `Elastic modulus GPa` / `弹性模量 GPa`：`207`
- `Density kg/m3` / `密度 kg/m3`：`7800`
- `Incident gauge distance mm` / `入射应变片距端 mm`：`839`
- `Transmitted gauge distance mm` / `透射应变片距端 mm`：`877`

4. 试样与符号
- `Section shape` / `截面形状`：选择 `cylinder`。
- `Specimen diameter mm` / `试样直径 mm`：`2.5`
- `Initial height/gauge length mm` / `初始高度/标距 mm`：`9.5`
- `Experiment type` / `实验类型`：选择 `tension`。
- `Sign convention` / `符号约定`：建议选择 `tension_positive`。

5. 处理与导出
- 保持 `Enable automatic fine alignment` / `启用自动微调对齐` 为勾选状态。
- 不勾选 `Enable standard SHPB wave separation` / `启用标准SHPB波形分离`，该数据按常规拉伸三波/两波流程处理。
- 点击 `Auto process` / `自动处理`。
- 复核 `Workflow status and result summary` / `流程状态与结果摘要`、`Three-wave alignment` / `三波对齐` 和 `Results` / `计算结果`。
- 若结果合理，点击 `Export Excel` / `导出 Excel` 或 `Generate report` / `生成报告` 导出结果。
