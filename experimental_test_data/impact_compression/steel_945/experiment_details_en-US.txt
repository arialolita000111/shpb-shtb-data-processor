# Experiment Details: 945 Steel Impact Compression

- Test type: impact compression
- Material: 945 steel
- Incident bar length: `2100 mm`
- Transmitted bar length: `1800 mm`
- Bar diameter: `20 mm`
- Specimen length: `4 mm`
- Specimen diameter: `8 mm`
- Bar elastic modulus: `210 GPa`
- Bar density: `7800 kg/m^3`
- Incident-bar strain gauge distance to specimen end face: `1020 mm`
- Transmitted-bar strain gauge distance to specimen end face: `770 mm`

## Software Input Guide

Applicable data files: `sample_01/raw_microstrain.xls`, `sample_02/raw_microstrain.xlsx`. Use `raw_voltage.xls` or `raw_voltage.xlsx` only if the voltage-to-microstrain calibration factor is known. Batch-processing examples are in `steel_945_compression_batch_input/`.

1. Data import
- Click `Import CSV/XLSX` / `导入 CSV/XLSX`, preferably choosing `raw_microstrain.xls` or `raw_microstrain.xlsx`.
- `Time column` / `时间列`: select `Time/s`.
- `Incident bar column` / `入射杆列`: select `incident_bar_signal/με`.
- `Transmitted bar column` / `透射杆列`: select `transmission_bar_signal/με`.

2. Acquisition and preprocessing
- `Time unit` / `时间单位`: select `s`.
- `Strain unit` / `应变单位`: select `με`.
- If you import a `raw_voltage` file, select `voltage` for `Strain unit` / `应变单位`, then enter the calibrated microstrain-per-volt factor in `Voltage conversion` / `电压换算`.
- `Baseline correction` / `基线修正`: beginners can keep the default `start_mean`.
- `Filter method` / `滤波方式`: beginners can start with `none`; try `moving_average` only if the waveform is visibly noisy.

3. Bar parameters
- `Incident bar diameter mm` / `入射杆直径 mm`: `20`
- `Transmitted bar diameter mm` / `透射杆直径 mm`: `20`
- `Elastic modulus GPa` / `弹性模量 GPa`: `210`
- `Density kg/m3` / `密度 kg/m3`: `7800`
- `Incident gauge distance mm` / `入射应变片距端 mm`: `1020`
- `Transmitted gauge distance mm` / `透射应变片距端 mm`: `770`

4. Specimen and sign convention
- `Section shape` / `截面形状`: select `cylinder`.
- `Specimen diameter mm` / `试样直径 mm`: `8`
- `Initial height/gauge length mm` / `初始高度/标距 mm`: `4`
- `Experiment type` / `实验类型`: select `compression`.
- `Sign convention` / `符号约定`: select `compression_positive`.

5. Processing and export
- Keep `Enable automatic fine alignment` / `启用自动微调对齐` checked.
- Do not check `Enable standard SHPB wave separation` / `启用标准SHPB波形分离`; this dataset uses the regular compression three-wave/two-wave workflow.
- Click `Auto process` / `自动处理`.
- Review `Workflow status and result summary` / `流程状态与结果摘要`, `Three-wave alignment` / `三波对齐`, and `Results` / `计算结果`.
- To process multiple files in `steel_945_compression_batch_input/`, first export batch parameters from one representative dataset, then use `Batch process with current parameters` / `按当前参数批处理`.
- If the results are reasonable, click `Export Excel` / `导出 Excel` or `Generate report` / `生成报告`.
