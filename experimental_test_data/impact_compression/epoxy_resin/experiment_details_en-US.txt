# Experiment Details: Epoxy Resin Impact Compression

- Test type: impact compression
- Material: epoxy resin
- Incident bar length: `2100 mm`
- Transmitted bar length: `1800 mm`
- Bar diameter: `20 mm`
- Specimen length: `4 mm`
- Specimen diameter: `10 mm`
- Bar elastic modulus: `210 GPa`
- Bar density: `7800 kg/m^3`
- Incident-bar strain gauge distance to specimen end face: `1018 mm`
- Transmitted-bar strain gauge distance to specimen end face: `741 mm`

## Software Input Guide

Applicable data files: `sample_01/raw_strain.csv`, `sample_02/raw_strain.xlsx`. Use `raw_voltage.csv` or `raw_voltage.xlsx` only if the voltage-to-strain calibration factor is known.

1. Data import
- Click `Import CSV/XLSX`, preferably choosing `raw_strain.csv` or `raw_strain.xlsx`.
- `Time column`: select `Time/s`.
- `Incident bar column`: select the incident-bar strain column, such as `incident_bar_signal/ε`.
- `Transmitted bar column`: select the transmitted-bar strain column, such as `transmission_bar_signal/ε`.

2. Acquisition and preprocessing
- `Time unit`: select `s`.
- `Strain unit`: select `strain`.
- If you import a `raw_voltage` file, select `voltage` for `Strain unit`, then enter the calibrated strain conversion factor in `Voltage conversion`.
- `Baseline correction`: beginners can keep the default `start_mean`.
- `Filter method`: beginners can start with `none`; try `moving_average` only if the waveform is visibly noisy.

3. Bar parameters
- `Incident bar diameter mm`: `20`
- `Transmitted bar diameter mm`: `20`
- `Elastic modulus GPa`: `210`
- `Density kg/m3`: `7800`
- `Incident gauge distance mm`: `1018`
- `Transmitted gauge distance mm`: `741`

4. Specimen and sign convention
- `Section shape`: select `cylinder`.
- `Specimen diameter mm`: `10`
- `Initial height/gauge length mm`: `4`
- `Experiment type`: select `compression`.
- `Sign convention`: select `compression_positive`.

5. Processing and export
- Keep `Enable automatic fine alignment` checked.
- Do not check `Enable standard SHPB wave separation`; this dataset uses the regular compression three-wave/two-wave workflow.
- Click `Auto process`.
- Review `Workflow status and result summary`, `Three-wave alignment`, and `Results`.
- If the results are reasonable, click `Export Excel` or `Generate report`.
